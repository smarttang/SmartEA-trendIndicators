## 实战指南

### 1. 时间框架选择

* **M1/M5：** 剥头皮最优选，我建议在这个时间框架进行打短线交易。
* **M30/H1/H4：** 中长线交易首选，因为目前我的 EA 就是基于这几个时间框架进行交易对冲，效果非常好。

### 2. 信号确认

* **多头 (Long)：** K 线保持蓝色，看板提示“做多趋势”。
* **空头 (Short)：** K 线保持红色，看板提示“做空趋势”。

### 3. 进场点位

当 K 线颜色发生反转的**第一根确认棒**即为高概率入场点位，但是不建议这个时间节点入场，我建议的时在趋势成立后超过 3 根 K 线确认后，进场交易，这个时候相对保守且盈利最优。

### 4. 风控管理

直接参考看板输出的：

* **建议止损位：** 保护您的本金安全。
* **建议止盈位：** 确保在利润回吐前锁定胜果。
